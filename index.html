ตั้งค่าเชื่อมต่อระบบคลาวด์ Firebase

การตั้งค่าข้อมูลนี้จะจัดเก็บไว้ที่คอมพิวเตอร์ของคุณเท่านั้น (ไม่แสดงในโค้ดต้นฉบับบน GitHub) ข้อมูลผู้สมัครจะเก็บแบบเรียลไทม์ไว้บนคลาวด์โปรเจกต์ของคุณเอง

ตั้งค่าคำที่คุณต้องการ เพื่อจำแนกกลุ่มข้อมูลบนคลาวด์ไม่ให้ซ้ำกับเว็บอื่น

🛠️ ขั้นตอนเปิดใช้งานคลาวด์ส่วนตัว:

1. สมัครใช้งาน Firebase Console และสร้าง Project ใหม่

2. เปิดใช้งาน Firestore Database และเปิดใช้ Anonymous Sign-in ในแท็บ Authentication

3. คัดลอกข้อมูลพารามิเตอร์ของ Web App ในโปรเจกต์คุณมาวางด้านบน
