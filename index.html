คุณแน่ใจหรือไม่ว่าจะลบรายการนี้? การลบจะเขียนลงระบบคลาวด์และไม่สามารถกู้คืนได้ภายหลัง

ตั้งค่าเชื่อมต่อฐานข้อมูล Firebase ของคุณ

ข้อมูลการเชื่อมต่อจะปลอดภัยจากการนำไปรันที่อื่น โดยจะจัดเก็บลงหน่วยความจำของบราวเซอร์บนอุปกรณ์เครื่องนี้โดยตรงเท่านั้น (ข้อมูลบน GitHub จะไม่ได้รับผลกระทบ)

ตั้งค่าชื่อโปรเจกต์ของคุณเอง เพื่อไม่ให้ผู้ใช้งานภายนอกเข้ามาทับข้อมูลชุดนี้ได้

🛠️ คัดลอกจาก Firebase Console ดังนี้:

1. ไปที่โครงการ Firebase Console ของคุณ

2. กดสัญลักษณ์เฟืองเกียร์ ⚙️ (Project Settings) ด้านซ้ายบน

3. เลื่อนลงมามองหาหัวข้อ "Your apps" ด้านล่างแล้วทำการก็อปปี้ config มากรอก
